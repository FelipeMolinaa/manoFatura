# Glossario

- `entity`: estrutura colocavel no grid
- `footprint`: area em celulas ocupada por uma entidade
- `placement`: ato de posicionar uma entidade no grid
- `worker`: trabalhador NPC
- `task_request`: pedido de trabalho enviado a um worker
- `task_result`: resultado de uma tarefa concluida, falha ou interrompida
- `slice`: entrega pequena e validavel
- `throughput`: volume de itens processados por unidade de tempo
